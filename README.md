# StatTest Selector

A guided statistical test selector for psychology and social science students.

**Live demo:** https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/

## Features

- 16 statistical tests with full explanations
- Smart Likert scale handling (single item vs composite scale)
- 3-step normality assessment with Shapiro-Wilk warning
- Side-by-side parametric vs non-parametric comparison
- APA 7th edition write-up templates with copy button
- SPSS, jamovi, and JASP step-by-step instructions
- G*Power paths for every test
- Decision breadcrumb + copy-to-clipboard methods paragraph
- Dark mode, animated transitions, tooltips

## Deploy to GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages → Source → GitHub Actions**
3. Push to `main` — the workflow builds and deploys automatically

## Run locally

\```bash
npm install
npm run dev
\```

## Tech stack

- React 18
- Vite 5
- lucide-react (icons)
- Inline styles only — no CSS framework required
